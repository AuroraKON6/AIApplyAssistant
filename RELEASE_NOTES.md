# Release Notes

## v1.5.0-beta

### 新功能

- **反检测基础设施**：6 个 JS 隐身脚本（webdriver 属性隐藏、navigator 覆盖、Chrome Runtime 模拟、AJAX 拦截、扩展绕过）+ uBlock Origin 扩展内置
- **Playwright 增强**：30+ Chromium 启动参数、中文语言环境、上海时区、CSP 绕过、自动化标志隐藏
- **页面健康恢复**：3 阶段健康检查 + 自动重建浏览器实例（PageRecoveryManager）
- **一键跨平台投递**：支持 BOSS直聘、智联招聘、前程无忧、猎聘四大平台顺序投递
- **任务执行管理**：每平台独立任务状态跟踪，支持终止执行（TaskExecutionManager）
- **AI 模板系统**：YAML 模板 + Mustache 渲染，结构化 Prompt 管理
- **输出校验**：长度验证、关键词覆盖率检查
- **新手引导**：简历解析 + 求职者画像生成
- **SSE 实时进度**：投递进度通过 Server-Sent Events 实时推送

### 改进

- 移除 Skyvern 浏览器 Agent 依赖，改用 Playwright 直接控制浏览器
- 重新启用各平台 /execute 和 /start 端点
- 模板化 Prompt 管理，避免硬编码
- 统一的平台枚举（RecruitmentPlatformEnum）和任务状态枚举（JobStatusEnum）

### 已知限制

- 部分网站可能检测到自动化操作并阻止访问
- 需要手机验证码或人脸验证的网站无法自动完成
- 网页搜索依赖外部搜索引擎，可能受网络环境影响

## v1.4.x-beta

### 改进

- 浏览器自动化稳定性提升
- 各平台投递流程优化

## v1.3.0-beta

### 新功能

- **平台分级投递**：优先返回 Greenhouse、Lever、Ashby 等 ATS 链接；Boss直聘标记为高风控并自动跳过。
- **浏览器资料隔离**：旧脚本使用独立浏览器资料目录，减少 Chromium 个人资料损坏弹窗。

### 改进

- 自动提交关闭时，任务完成后显示"等待确认"，不再误显示为已提交。
- 损坏的浏览器资料目录会自动备份并重建，必要时提示用户重新登录。

## v1.2.0

### 新功能

- **岗位发现 Agent**：输入求职目标后，AI 联网搜索真实岗位，返回带可信度和来源证据的候选列表
- **真实联网搜索**：通过 Bing / DuckDuckGo 搜索引擎获取真实岗位链接，非 LLM 编造
- **运行诊断**：一键检查 AI 配置、API 连接、Playwright 引擎、简历文件、网页搜索是否正常
- **保存并应用**：AI 配置保存后立即生效，不需要重启软件
- **自动提交申请开关**：默认关闭，关闭时 AI 填写表单后暂停等用户确认

### 改进

- 搜索引擎优先使用 Bing（中国大陆更稳定），DuckDuckGo 作为备选
- 岗位搜索结果按 URL 去重，防止重复投递
- LLM 返回的 URL 会与实际搜索结果交叉验证，过滤编造链接
- 简化前端配置流程，减少不必要的重启提示

### 已知限制

- 本版本为 **测试版**，不建议无人值守自动投递
- 需要手机验证码、人脸验证的网站无法自动完成
- 部分网站可能检测到自动化操作并阻止访问
- 网页搜索依赖外部搜索引擎，可能受网络环境影响
- 投递成功率取决于目标网站的反自动化策略
