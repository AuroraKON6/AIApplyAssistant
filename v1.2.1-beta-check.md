## 需要手动验证的项目

以下项目需要用户使用真实 API Key 手动测试：

- [ ] 保存真实 AI 配置（模型名、API 地址、API Key）后，点击"保存并应用"
- [ ] 不重启软件，直接点击"先查找岗位"
- [ ] 输入"上海 Python 后端实习"，确认返回真实可打开的岗位链接
- [ ] 勾选 1 个岗位，关闭"自动提交申请"
- [ ] 点击"投递选中岗位"，确认：
  - 浏览器自动打开目标页面
  - AI 填写申请表单
  - 上传简历（如已配置）
  - 在提交按钮前停止，等待用户确认

## 已知限制

- 网页搜索依赖 Bing/DuckDuckGo，中国大陆网络环境下 DuckDuckGo 可能不可用
- 需要手机验证码/人脸验证的网站无法自动完成
- 部分网站可能检测到自动化操作并阻止访问
- 本版本为测试版，不建议无人值守自动投递
