# AI 投递助手

基于 Playwright 浏览器自动化的智能求职投递工具。AI 先帮你搜索真实岗位，你确认后一键自动填写申请，支持 BOSS直聘、智联招聘、前程无忧、猎聘四大平台。

## 快速开始

### 1. 下载并解压

从 Release 页面下载 `AIApplyAssistant-v1.5.0-beta.zip`，解压到任意目录。

### 2. 启动

双击 `start_ai_apply.bat`（Windows）或在终端运行：

```bash
java -jar app.jar
```

启动后浏览器会自动打开管理页面（默认 http://localhost:8888/ai-apply）。

### 3. 配置 AI 模型

在页面顶部找到 **AI 配置** 区域，填写：

| 字段 | 说明 | 示例 |
|------|------|------|
| AI模型名称 | 你要使用的模型 | `deepseek-chat`、`gpt-4o`、`mimo-v2.5-pro` |
| API 地址 | OpenAI 兼容的 API 地址 | `https://api.deepseek.com/v1` |
| API Key | 你的 API 密钥 | `sk-xxx` |

填写完成后点击 **保存并应用** 按钮。配置会立即生效，不需要重启软件。

> API Key 只保存在本地，不会上传到任何服务器。

### 4. 运行诊断

点击页面上的 **开始检测** 按钮，检查以下组件是否正常：

- AI 模型配置 — 是否已填写完整
- AI 模型连接 — API 是否可达
- Playwright 浏览器引擎 — Chromium 是否可用
- 简历文件 — 是否存在
- 网页搜索 — 搜索引擎是否可用

全部显示绿色勾号即可开始使用。

### 5. 查找岗位

1. 在 **求职目标** 文本框中描述你想找的工作，例如："上海 Python 后端实习"
2. （可选）在 **公司名单** 中粘贴你感兴趣的公司名称，每行一个
3. 点击 **先查找岗位**
4. AI 会联网搜索真实岗位，返回候选列表
5. 每个岗位会显示可信度（高可信 / 中可信 / 待确认）和来源证据

### 6. 投递岗位

1. 勾选你想投递的岗位
2. 确认 **自动提交申请** 开关状态：
   - **关闭**（默认）：AI 会填写申请表单，但在提交前暂停，等你确认
   - **开启**：AI 会自动完成提交
3. 点击 **投递选中岗位**
4. 浏览器会自动打开、填写信息、上传简历。自动提交关闭时会停在最终提交前，由你确认。

## 核心功能

### 一键跨平台投递

支持四大招聘平台的统一流水线投递：

| 平台 | 说明 |
|------|------|
| BOSS直聘 | 支持岗位搜索和自动投递 |
| 智联招聘 | 支持岗位搜索和自动投递 |
| 前程无忧/51job | 支持岗位搜索和自动投递 |
| 猎聘 | 支持岗位搜索和自动投递 |

跨平台投递采用顺序执行模式，每个平台独立任务状态跟踪，支持随时终止。

### 反检测体系

内置多层反检测机制，降低被招聘网站识别为自动化的风险：

- **隐身脚本注入**：webdriver 属性隐藏、navigator 属性覆盖、Chrome Runtime 模拟、AJAX 拦截
- **uBlock Origin 扩展**：内置广告拦截扩展，模拟真实用户浏览器环境
- **Playwright 增强配置**：30+ Chromium 启动参数、中文语言环境、上海时区、CSP 绕过
- **页面健康恢复**：3 阶段健康检查 + 自动重建浏览器实例

### AI 模板系统

结构化的 Prompt 管理，支持 YAML 模板和 Mustache 渲染：

- 模板化 Prompt 管理，避免硬编码
- 输出校验：长度验证、关键词覆盖率检查
- 支持自定义模板扩展

### 新手引导

首次使用时的引导流程：

- 解析用户简历，自动提取关键信息
- 生成求职者画像
- 辅助配置求职目标

## 关于自动提交申请

**默认关闭。** 关闭时，AI 填写完表单后会暂停在提交按钮前，等你确认后再提交。

开启自动提交前请注意：
- 自动提交可能违反某些招聘网站的使用条款
- 提交后无法撤回
- 建议先用"关闭"模式观察几次，确认行为符合预期后再开启

## Excel 公司名单

你可以上传一个 Excel 文件，其中包含你想投递的公司名称。AI 会优先搜索这些公司的岗位。

这只是一个**可选线索**，AI 也会自行搜索其他匹配的岗位。

## 关于投递状态

- **执行中**：浏览器正在自动操作
- **等待确认**：自动提交关闭时，已经准备到提交前，需要你手动确认
- **已提交**：自动提交开启且任务结果明确提交完成
- **失败**：任务执行失败或被平台阻止

## 已知限制

- 部分网站可能检测到自动化操作并阻止访问
- 需要手机验证码或人脸验证的网站无法自动完成
- 网页搜索依赖外部搜索引擎，可能受网络环境影响
- 本工具不保证投递成功率
- 如果 Chromium 提示个人资料损坏，程序会备份损坏的资料目录并新建干净资料目录，可能需要重新登录招聘网站

## 技术栈

- **后端**: Spring Boot + MyBatis + SQLite
- **前端**: Next.js + Tailwind CSS
- **浏览器自动化**: Playwright (Java)
- **AI**: OpenAI 兼容 API（支持 DeepSeek、GPT-4o 等）
