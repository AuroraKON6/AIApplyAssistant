# 隐私说明

## 数据存储

所有用户数据**仅保存在本地**：

- **API Key**：保存在本地配置文件中，不会上传到任何服务器
- **简历文件**：保存在用户本地磁盘，不会上传（除非用户主动投递到招聘网站）
- **运行数据**：投递记录、Cookie 等保存在本地 SQLite 数据库中
- **AI 配置**：保存在本地数据库，不会同步到云端

## 发布包

发布包（zip 文件）**不包含**以下敏感文件：

- `.env` — API Key 等配置
- `.db` / `.sqlite` — 运行数据库
- `.log` — 运行日志
- 用户简历文件

用户首次运行时需要自行配置 AI 模型和简历路径。

## 网络请求

本工具会在运行时发起以下网络请求：

- **AI 模型 API**：调用用户配置的 LLM 服务（如 DeepSeek、OpenAI）
- **搜索引擎**：通过 Bing / DuckDuckGo 搜索岗位信息
- **招聘网站**：通过 Playwright 浏览器自动化访问用户指定的招聘网站

所有请求均在用户本地发起，不经过任何中间服务器。

## 用户责任

使用本工具前，请注意：

- 用户需自行确认所使用招聘网站的规则和条款
- 自动投递行为可能违反某些网站的使用条款
- 用户需自行承担因使用本工具而产生的任何风险
- 本工具不保证投递成功率或岗位匹配准确度

## 自动提交申请

**默认关闭。** 开启后，AI 会自动完成申请表单的提交。

风险提示：
- 提交后无法撤回
- 可能违反招聘网站的使用条款
- 建议先用"关闭"模式观察几次，确认行为符合预期后再开启
