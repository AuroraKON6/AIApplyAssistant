# AI 投递助手

基于 Skyvern 浏览器 Agent 的自动求职投递工具。AI 先帮你搜索真实岗位，你确认后再自动填写申请。

## 快速开始

### 1. 下载并解压

从 Release 页面下载 `AIApplyAssistant-v1.3.0-beta.zip`，解压到任意目录。

### 2. 启动

双击 `start_ai_apply.bat`（Windows）或在终端运行：

```bash
java -jar app.jar
```

启动后浏览器会自动打开管理页面（默认 http://localhost:8888/ai-apply）。

### 3. 配置 AI 模型

在页面顶部找到 **AI 配置** 区域，填写：

| 字段 | 说明 | 示例 |
|------|------|------|
| AI模型名称 | 你要使用的模型 | `deepseek-chat`、`gpt-4o`、`mimo-v2.5-pro` |
| API 地址 | OpenAI 兼容的 API 地址 | `https://api.deepseek.com/v1` |
| API Key | 你的 API 密钥 | `sk-xxx` |

填写完成后点击 **保存并应用** 按钮。配置会立即生效，不需要重启软件。

> API Key 只保存在本地 `runtime/skyvern/.env` 文件中，不会上传到任何服务器。

### 4. 运行诊断

点击页面上的 **开始检测** 按钮，检查以下组件是否正常：

- AI 模型配置 — 是否已填写完整
- AI 模型连接 — API 是否可达
- 浏览器执行服务 (Skyvern) — 是否运行中
- Playwright 浏览器引擎 — Chromium 是否可用
- 简历文件 — 是否存在
- 网页搜索 — 搜索引擎是否可用

全部显示绿色勾号即可开始使用。

### 5. 查找岗位

1. 在 **求职目标** 文本框中描述你想找的工作，例如："上海 Python 后端实习"
2. （可选）在 **公司名单** 中粘贴你感兴趣的公司名称，每行一个
3. 点击 **先查找岗位**
4. AI 会联网搜索真实岗位，返回候选列表
5. 每个岗位会显示可信度（高可信 / 中可信 / 待确认）和来源证据

### 6. 投递岗位

1. 勾选你想投递的岗位
2. 确认 **自动提交申请** 开关状态：
   - **关闭**（默认）：AI 会填写申请表单，但在提交前暂停，等你确认
   - **开启**：AI 会自动完成提交
3. 点击 **投递选中岗位**
4. 浏览器会自动打开、填写信息、上传简历。自动提交关闭时会停在最终提交前，由你确认。

## 关于自动提交申请

**默认关闭。** 关闭时，AI 填写完表单后会暂停在提交按钮前，等你确认后再提交。

开启自动提交前请注意：
- 自动提交可能违反某些招聘网站的使用条款
- 提交后无法撤回
- 建议先用"关闭"模式观察几次，确认行为符合预期后再开启

## Excel 公司名单

你可以上传一个 Excel 文件，其中包含你想投递的公司名称。AI 会优先搜索这些公司的岗位。

这只是一个**可选线索**，AI 也会自行搜索其他匹配的岗位。

## 平台支持情况

| 平台 | 通道类型 | 投递难度 | 说明 |
|------|----------|----------|------|
| Greenhouse | ATS | 可自动投递 | 多数海外科技公司使用，支持表单填写和简历上传 |
| Lever | ATS | 可自动投递 | 初创公司常用，ATS 系统标准化程度高 |
| Ashby | ATS | 可自动投递 | 现代 ATS，表单结构清晰 |
| 公司官网招聘页 | 公司官网 | 可能需要登录 | 取决于具体网站的反爬策略 |
| 智联招聘 | 招聘平台 | 可能需要登录 | 国内主流平台，可能触发验证码 |
| 猎聘 | 招聘平台 | 可能需要登录 | 国内主流平台，可能需要登录 |
| 前程无忧/51job | 招聘平台 | 可能需要登录 | 国内主流平台，可能触发风控 |
| Boss直聘 | 招聘平台 | 高风控 | 反自动化检测严格，不支持自动投递，需手动处理 |

### 关于投递状态

- **执行中**：浏览器正在自动操作
- **等待确认**：自动提交关闭时，已经准备到提交前，需要你手动确认
- **已提交**：自动提交开启且任务结果明确提交完成
- **失败**：任务执行失败或被平台阻止

> `queued` 状态只表示任务已创建，不代表投递成功。自动提交关闭时，即使 Skyvern 任务完成，页面也会显示"等待确认"，不会直接显示为已提交。

### 推荐使用方式

1. **优先投递**：ATS 平台（Greenhouse、Lever、Ashby）— 自动化成功率最高
2. **可以尝试**：智联招聘、猎聘、前程无忧 — 可能需要人工介入（登录、验证码）
3. **需手动处理**：Boss直聘 — 系统会自动跳过并提示

## 已知限制

- 部分网站可能检测到自动化操作并阻止访问
- 需要手机验证码或人脸验证的网站无法自动完成
- 网页搜索依赖外部搜索引擎，可能受网络环境影响
- 本工具不保证投递成功率
- Skyvern 任务可能因页面复杂、登录、验证码或风控而需要人工接管
- 如果 Chromium 提示个人资料损坏，程序会备份损坏的资料目录并新建干净资料目录，可能需要重新登录招聘网站
