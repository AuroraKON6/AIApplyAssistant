# Release Notes

## v1.3.0-beta

### 新功能

- **平台分级投递**：优先返回 Greenhouse、Lever、Ashby 等 ATS 链接；Boss直聘标记为高风控并自动跳过。
- **Skyvern 查看入口**：Skyvern 任务返回任务接口、浏览器调试入口、录屏/截图入口（如服务返回），前端可直接查看执行情况。
- **浏览器资料隔离**：旧脚本和 Skyvern 使用独立浏览器资料目录，减少 Chromium 个人资料损坏弹窗。

### 改进

- 自动提交关闭时，任务完成后显示"等待确认"，不再误显示为已提交。
- Skyvern prompt 明确要求：如果按钮会直接提交，必须停在点击前。
- 损坏的浏览器资料目录会自动备份并重建，必要时提示用户重新登录。
- 智联招聘保留原生脚本投递通道；公司官网和 ATS 走 Skyvern。

### 已知限制

- Skyvern 是否能直接显示浏览器窗口取决于本地浏览器和 Skyvern 运行方式；如果窗口不可见，请使用页面里的任务接口、截图、录屏或浏览器调试入口。
- 招聘平台登录、验证码、短信验证、风控弹窗仍需要人工处理。
- Boss直聘暂不支持自动投递。

## v1.2.0

### 新功能

- **岗位发现 Agent**：输入求职目标后，AI 联网搜索真实岗位，返回带可信度和来源证据的候选列表
- **真实联网搜索**：通过 Bing / DuckDuckGo 搜索引擎获取真实岗位链接，非 LLM 编造
- **运行诊断**：一键检查 AI 配置、API 连接、Skyvern 服务、Playwright 引擎、简历文件、网页搜索是否正常
- **保存并应用**：AI 配置保存后立即生效，不需要重启软件
- **自动提交申请开关**：默认关闭，关闭时 AI 填写表单后暂停等用户确认
- **活跃任务保护**：保存配置时如有正在投递的任务，不会强制重启 Skyvern

### 改进

- 搜索引擎优先使用 Bing（中国大陆更稳定），DuckDuckGo 作为备选
- 岗位搜索结果按 URL 去重，防止重复投递
- LLM 返回的 URL 会与实际搜索结果交叉验证，过滤编造链接
- 简化前端配置流程，减少不必要的重启提示

### 已知限制

- 本版本为 **测试版**，不建议无人值守自动投递
- 需要手机验证码、人脸验证的网站无法自动完成
- 部分网站可能检测到自动化操作并阻止访问
- 网页搜索依赖外部搜索引擎，可能受网络环境影响
- 投递成功率取决于目标网站的反自动化策略
